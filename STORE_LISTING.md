## App Privacy answers (App Store Connect → App Privacy)

Answer the questionnaire exactly like this — it must match what the app really does.

**Data used to track you:** None.

**Data linked to you (tied to account/user ID):**
| Data type | Purpose | Notes |
|---|---|---|
| Email address | App Functionality | Account/auth |
| Name | App Functionality | Display name |
| Photos (optional avatar) | App Functionality | Only if user sets one |
| User content (community messages) | App Functionality | Public chat rooms |
| Purchases / subscription status | App Functionality | Via RevenueCat/Apple |
| User ID | App Functionality | Supabase auth id |

**Data linked to you — for AI:**
| Data type | Purpose |
|---|---|
| User content (messages you type to the AI coach) | App Functionality |

**Data NOT collected (stays on device — declare as not collected):**
- Health/craving logs, statistics, "The Room" conversations, app settings.

**Data used for tracking / advertising:** None. No third-party ad SDKs.

> Rule of thumb: if the data leaves the device it's "collected"; on-device-only data is not. Logs and The Room are on-device → not collected.
